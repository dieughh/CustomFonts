# Custom Fonts

Скрипт для полной кастомизации шрифтов в интерфейсе Яндекс Музыки. Позволяет не только менять стандартный шрифт на любой пользовательский, но и тонко настраивать его визуальные параметры.

# Особенности

- Загрузка своих шрифтов: Поддержка файлов форматов .ttf, .otf, .woff и .woff2.

- Гибкая типографика: Настройка жирности и межбуквенного расстояния с помощью ползунков.

- Стили текста: Быстрое включение курсива, подчеркивания или зачеркивания текста.

- Эффекты тени: Создание объема для текста с настройкой смещения, размытия и цвета тени.

- Надежное хранение: Шрифты сохраняются в базе данных IndexedDB, поэтому их не нужно загружать заново после перезапуска.

# Как пользоваться

1. Откройте настройки: Перейдите в раздел настроек внутри приложения Яндекс Музыка.

2. Откройте меню настройки шрифтов (кнопка "Кастомный шрифт")

3. Загрузите шрифт: Нажмите на кнопку выбора шрифта в интерфейсе и выберите файл на вашем компьютере.

4. Настройте стиль: Настройте внешний вид шрифта при помощи настроек в Pulse Sync.

5. Сброс: Чтобы вернуть стандартный шрифт, просто воспользуйтесь кнопкой "Сбросить шрифт" в меню аддона.
